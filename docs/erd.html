Hook-Driven Side Effects

  • VT Task.on_submit → point_calculator.calculate_points → inserts Task Point Log rows → updates User Point Summary.
  • VT Task.on_update → scheduling_engine.on_task_update + analytics.invalidate_project_cache.
  • VT Project.validate → vt_project.validate_team.
  • VT Sprint.on_update → analytics cache invalidation.
  • Notification Log.after_insert → push_sender.send_push_for_notification → filters via Vernon Push Preference → dispatches to all Vernon Push Subscription rows for user → prunes dead endpoints (404/410).
