TopNav Dual Navbar Redesign

Date: 2026-05-21  |  Status: Approved

Scope

Redesign desktop navigation (TopNav.tsx) from a single-row tab bar into a two-row navbar. Change post-login redirect from /m/work to /m/dashboard. Increase content padding. No icon library — use inline SVG flat icons.

Navbar1 (top row, height 48px)

[VERNON logo]  |  [Breadcrumb]  ·····spacer·····  [bell icon]  [avatar icon]
  • Logo: "VERNON" text, links to /m/dashboard, Barlow Condensed font, blue accent.
  • Divider: 1px vertical line, muted color.
  • Breadcrumb: derived from active route — maps path to label (Dashboard, Work, Analytics, Me, Leader).
  • Spacer: flex-grow pushes right icons.
  • Bell icon: flat inline SVG; click → dropdown overlay showing unread notifications (reuse NotificationBell + useNotificationCount hooks). Badge on icon when unread > 0.
  • Avatar icon: flat inline SVG person icon; click → dropdown with Profil link (/m/me) and Logout button (calls frappe.call logout).

Navbar2 (second row, height 40px, bg #f1f5f9)

[Dashboard]  [Project]  [Report]
  • Hardcoded items — no DocType fetch.
  • Dashboard → /m/dashboard
  • Project → /m/work
  • Report → /m/analytics
  • Active item: border-bottom 2px blue, text color dark.
  • Background: #f1f5f9 (light grey).

Responsiveness

  • ≥768px: show navbar1 + navbar2. Hide BottomNav.
  • <768px: hide both navbars. Show BottomNav (unchanged).
  • Navbar2 scrolls horizontally on narrow desktop if items overflow.

Padding

  • .db-root: 20px 22px → 28px 36px
  • @media max-width 767px: 12px → 16px

Login Redirect

  • router.tsx OnboardingGate: /m/work → /m/dashboard
  • router.tsx catch-all *: /m/work → /m/dashboard
  • login.html JS default redirect: /m/work → /m/dashboard

Files Changed

  1. pwa/src/components/TopNav.tsx — full redesign
  2. pwa/src/theme/index.ts — add topnav CSS tokens
  3. pwa/src/portal/dashboard/dashboard.css — padding increase
  4. pwa/src/router.tsx — redirect targets
  5. vernon_tasks/www/login.html — default redirect

Out of Scope

  • BottomNav changes
  • Portal (/portal/*) TopBar
  • Mobile Dashboard content changes
