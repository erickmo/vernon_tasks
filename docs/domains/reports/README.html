Scope

Reporting hub. Aggregates analytics, OKR rollup, sprint velocity, risk, leaderboard, workload, and overdue tasks. Originally exposed via two surfaces (both removed — desk-only migration); the aggregation now backs the desk analytics pages:

  • /portal/reports — desktop exec dashboard (3 tabs: OKR, Sprints, Team). (removed — desk-only migration; superseded by /app/exec-analytics + /app/leader-analytics)
  • /m/reports — mobile hub (3 cards: My Reports, Projects I Manage, My Team). (removed — desk-only migration; superseded by /app/my-analytics)
