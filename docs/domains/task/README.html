All rules in doctype controllers — Frappe Hooks-First per ADR-022.

HookRule

Task Schedule Entry

Rule
allocated_minutes ∈ (0, 1440]; minutes_planned ∈ [0, 1440]; hour_start ∈ [0, 23]

Task Dependency

Rule
blocked_by required, must reference an existing VT Task; dependency_type ∈ {Finish-to-Start, Start-to-Start}; self-block detected at parent

Task Point Log (append-only ledger)

Rule
amount sign matches transaction_type: earned/early_bonus > 0; late_penalty/revision_deduction < 0; leader_override ≠ 0
log_timestamp ≤ now
leader_override requires overridden_by + note
Rows are immutable once inserted (task/user/type/amount/timestamp can't change)

Risk Event

Rule
reason normalize + ≤ 140 char; severity ∈ {high, med, low}; resolved_at ≥ detected_at; task.project must equal risk_event.project

Recurring Rule

Rule
rule_type ∈ {Daily, Weekly, Monthly, Custom}; interval ∈ [1, 365]; day_of_month ∈ [1, 31]; days_of_week ⊆ {Mon..Sun}
max_occurrences ≥ 1; end_date ≥ today on create

VT Report Subscription

Rule
slug ∈ ALLOWED_SLUGS (whitelist); cron is 5-field expression; filters_json must parse as JSON; recipients non-empty + unique; each recipient.user must exist
